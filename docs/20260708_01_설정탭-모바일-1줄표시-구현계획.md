# 설정탭 모바일 1줄표시 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 모바일 설정 모달 상단의 4개 탭(`표시옵션`, `초기화면`, `태그관리`, `저장갱신`)이 360px 폭에서 글자 크기 유지 상태로 1줄 표시된다.

**Architecture:** 기존 `SettingsModal.tsx` 구조를 유지하고, 모바일 전용 Tailwind class만 조정한다. 모달 최대 폭 `max-w-[90vw]`, 탭 글자 크기 `text-sm`, 탭 위아래 여백 `py-2`는 유지한다. 320px급 초소형 화면에서는 줄바꿈 대신 가로 스크롤 fallback으로 잘림을 막는다.

**Tech Stack:** React, TypeScript, Tailwind CSS CDN, Vite, Playwright

---

## 1. 확정 범위

| 항목 | 결정 | 근거 |
|---|---|---|
| 글자 크기 | ✅ `text-sm` 유지 | 사용자가 글자 축소를 원하지 않음 |
| 탭 위아래 여백 | ✅ `py-2` 유지 | 터치 높이를 불필요하게 줄이지 않음 |
| 탭 좌우 여백 | 🔄 모바일 `px-4` -> `px-2` | 4개 탭 총폭을 줄여 360px에서 1줄 확보 |
| 모달 안쪽 여백 | 🔄 모바일 `p-6` -> `p-4` | 탭이 사용할 수 있는 가로 공간 확보 |
| 모달 최대 폭 | ✅ `max-w-[90vw]` 유지 | 미리보기의 `92vw` 확장은 최종 권장안에서 제외 |
| 320px fallback | ⚠️ 탭 컨테이너 `overflow-x-auto` | 작은 화면에서 wrap/cut 대신 스크롤 허용 |

## 4. 구현 정책

### 4.1 모달 class

현재:

```tsx
<div className="bg-elevated shadow-2xl rounded-lg p-6 w-[600px] max-w-[90vw] max-h-[85vh] overflow-hidden flex flex-col">
```

변경:

```tsx
<div className="bg-elevated shadow-2xl rounded-lg p-4 sm:p-6 w-[600px] max-w-[90vw] max-h-[85vh] overflow-hidden flex flex-col">
```

완료 기준:

- 640px 미만에서는 모달 안쪽 여백이 16px이다.
- 640px 이상에서는 기존 24px 여백이 유지된다.
- `max-w-[90vw]`는 변경되지 않는다.

### 4.2 탭 컨테이너 class

현재:

```tsx
<div className="flex border-b border-secondary mb-6">
```

변경:

```tsx
<div className="flex overflow-x-auto border-b border-secondary mb-6">
```

완료 기준:

- 360px에서는 가로 스크롤이 발생하지 않고 4개 탭이 한눈에 보인다.
- 320px처럼 폭이 더 작은 화면에서는 텍스트가 접히거나 잘리지 않고, 필요 시 탭 영역만 가로 스크롤된다.

### 4.3 탭 버튼 class

현재 4개 버튼 공통 패턴:

```tsx
className={`px-4 py-2 text-sm font-medium border-b-2 ${
  activeTab === 'display'
    ? 'border-focus text-blue-600'
    : 'border-transparent text-secondary hover:text-primary'
}`}
```

변경 4개 버튼 공통 패턴:

```tsx
className={`px-2 sm:px-4 py-2 text-sm font-medium border-b-2 whitespace-nowrap ${
  activeTab === 'display'
    ? 'border-focus text-blue-600'
    : 'border-transparent text-secondary hover:text-primary'
}`}
```

각 버튼의 `activeTab` 비교값은 기존 값을 유지한다.

| 탭 | 기존 비교값 | 변경 여부 |
|---|---|---|
| 표시옵션 | `activeTab === 'display'` | ✅ 유지 |
| 초기화면 | `activeTab === 'initial'` | ✅ 유지 |
| 태그관리 | `activeTab === 'tags'` | ✅ 유지 |
| 저장갱신 | `activeTab === 'data'` | ✅ 유지 |

완료 기준:

- 모바일에서 좌우 padding은 8px이다.
- `sm` 이상에서는 기존 좌우 padding 16px이다.
- `py-2`, `text-sm`, `font-medium`, `border-b-2`는 유지된다.
- 모든 탭 텍스트는 1줄이다.
