# 자동태그 계정별 규칙 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 로그인 사용자별 태그 설정에 자동 키워드 규칙을 추가하고, 새 책과 기존 책에 자동태그를 계산해 수동 태그와 분리 관리한다.

**Architecture:** 자동 규칙은 기존 `user_settings.settings.tagSettings` 안에 계정별로 저장한다. 책별 자동 적용 결과는 `user_library.book_data.autoTags`에 저장하고, 기존 `customTags`는 수동/레거시 태그로 보존한다. 화면과 필터는 `customTags + autoTags`의 병합 결과를 사용하되, 자동 규칙 재계산은 `autoTags`만 덮어쓴다.

**Tech Stack:** React, TypeScript, Zustand, Supabase, Vite

---

## 확정된 요구사항

- 자동태그 규칙은 계정별 설정이다.
- 기본 자동 키워드는 기존 태그명 1개로 자동 생성한다.
- 새 책 추가 시 자동태그를 계산해 함께 저장한다.
- 기존 책에도 자동태그 재계산을 적용한다.
- 자동태그는 자동 추가와 자동 제거가 모두 가능해야 한다.
- 기존 `customTags`는 초기 구현에서 이동하지 않고 그대로 보존한다.
- `customTags`와 `autoTags`가 중복되면 화면에는 하나만 보인다.
- 사용자가 태그 키워드를 수동으로 추가/삭제할 수 있어야 한다.

## 포함하지 않는 범위

- 관리자 계정 규칙을 다른 계정에 강제 배포하지 않는다.
- 레거시 `customTags`를 자동으로 `autoTags`로 이동하지 않는다.
- AI 추천/외부 API 기반 키워드 생성은 하지 않는다.
- 제목 외 저자/출판사/소개 매칭은 1차 구현에서 제외한다.
- `읽는중`, `오디오북`, `my보유` 같은 상태성 태그를 특별 처리해 자동 비활성화하지 않는다. 초기 기본값은 태그명 1개로 만들되 사용자가 끌 수 있게 한다.

## 저장 데이터 영향

영향받는 저장 대상은 Supabase `user_settings.settings`와 `user_library.book_data`이다.

- `user_settings.settings.tagSettings.autoTagRules`를 추가한다.
- 기존 사용자의 `tagSettings.autoTagRules`가 없으면 런타임에서 `tags[]`를 기준으로 기본 규칙을 보강한다.
- 신규 태그 생성 시 같은 이름의 자동 키워드 1개를 가진 규칙을 함께 만든다.
- 태그 삭제 시 해당 태그의 자동 규칙도 삭제한다.
- 태그 이름 변경 시 사용자가 규칙을 따로 수정하지 않았다면 기본 키워드도 새 태그명으로 동기화한다.
- `user_library.book_data.autoTags`를 추가한다.
- 기존 `customTags`는 그대로 둔다.
- 자동태그 재계산은 `autoTags`만 업데이트한다.

검증 기준:

- 기존 데이터에 `autoTagRules`가 없어도 설정 화면이 깨지지 않는다.
- 기존 데이터에 `autoTags`가 없어도 서재 목록/상세/필터가 깨지지 않는다.
- 새 책 추가 시 제목에 태그명 키워드가 있으면 `autoTags`에 해당 태그 id가 저장된다.
- 규칙에서 키워드를 제거하면 기존 책 재계산 후 해당 자동태그가 제거된다.
- 같은 태그가 `customTags`와 `autoTags`에 동시에 있어도 화면에는 1개만 보인다.

```ts
import type { AutoTagRule, CustomTag, SelectedBook } from '../types';

const normalizeText = (value: string): string =>
  value
    .toLocaleLowerCase('ko-KR')
    .normalize('NFKC')
    .replace(/\s+/g, ' ')
    .trim();

const normalizeKeyword = (keyword: string): string => normalizeText(keyword);

const dedupeKeywords = (keywords: string[]): string[] => {
  const seen = new Set<string>();
  const result: string[] = [];

  keywords.forEach(keyword => {
    const trimmed = keyword.trim();
    const normalized = normalizeKeyword(trimmed);
    if (!normalized || seen.has(normalized)) return;
    seen.add(normalized);
    result.push(trimmed);
  });

  return result;
};

export const normalizeAutoTagRules = (
  tags: CustomTag[],
  rules: AutoTagRule[] | undefined,
): AutoTagRule[] => {
  const now = Date.now();
  const tagIds = new Set(tags.map(tag => tag.id));
  const ruleMap = new Map((rules ?? []).filter(rule => tagIds.has(rule.tagId)).map(rule => [rule.tagId, rule]));

  return tags.map(tag => {
    const existing = ruleMap.get(tag.id);
    if (existing) {
      const keywords = dedupeKeywords(existing.keywords);
      return {
        ...existing,
        enabled: existing.enabled,
        keywords: keywords.length > 0 ? keywords : [tag.name],
        matchFields: ['title'],
      };
    }

    return {
      tagId: tag.id,
      enabled: true,
      keywords: [tag.name],
      matchFields: ['title'],
      updatedAt: now,
    };
  });
};

export const getMergedTagIds = (book: Pick<SelectedBook, 'customTags' | 'autoTags'>): string[] =>
  Array.from(new Set([...(book.customTags ?? []), ...(book.autoTags ?? [])]));

export const calculateAutoTagIds = (
  book: Pick<SelectedBook, 'title'>,
  rules: AutoTagRule[],
): string[] => {
  const normalizedTitle = normalizeText(book.title ?? '');

  return rules
    .filter(rule => rule.enabled)
    .filter(rule => rule.keywords.some(keyword => normalizedTitle.includes(normalizeKeyword(keyword))))
    .map(rule => rule.tagId);
};

export const parseKeywordInput = (value: string): string[] =>
  dedupeKeywords(value.split(','));
```

updateAutoTagRule: async (tagId, updates) => {
  const currentSettings = get().settings;
  const currentRules = normalizeAutoTagRules(
    currentSettings.tagSettings.tags,
    currentSettings.tagSettings.autoTagRules,
  );

  const updatedRules = currentRules.map(rule =>
    rule.tagId === tagId
      ? {
          ...rule,
          ...updates,
          keywords: updates.keywords ? parseKeywordInput(updates.keywords.join(',')) : rule.keywords,
          updatedAt: Date.now(),
        }
      : rule
  );

  const updatedTagSettings = {
    ...currentSettings.tagSettings,
    autoTagRules: updatedRules,
  };

  await get().updateUserSettings({ tagSettings: updatedTagSettings });
},

```ts
recalculateAutoTagsForAllBooks: async () => {
  const { settings } = useSettingsStore.getState();
  const rules = settings.tagSettings.autoTagRules ?? [];
  const { isAllBooksLoaded, fetchRemainingLibrary } = get();

  if (!isAllBooksLoaded) {
    await fetchRemainingLibrary();
  }

  const books = get().myLibraryBooks;

  let changed = 0;

  const updates = books.map(async (book) => {
    const nextAutoTags = calculateAutoTagIds(book, rules);
    const current = JSON.stringify([...(book.autoTags ?? [])].sort());
    const next = JSON.stringify([...nextAutoTags].sort());
    if (current === next) return null;

    changed += 1;
    const updatedBook = { ...book, autoTags: nextAutoTags };
    const { id, ...bookDataForDb } = updatedBook;

    const { error } = await supabase
      .from('user_library')
      .update({
        title: updatedBook.title,
        author: updatedBook.author,
        book_data: bookDataForDb as unknown as Json,
      })
      .eq('id', id);

    if (error) throw error;
    return updatedBook;
  });

  const results = (await Promise.all(updates)).filter((book): book is SelectedBook => Boolean(book));
  const updatedMap = new Map(results.map(book => [book.id, book]));

  set(state => ({
    myLibraryBooks: state.myLibraryBooks.map(book => updatedMap.get(book.id) ?? book),
    librarySearchResults: state.librarySearchResults.map(book => updatedMap.get(book.id) ?? book),
    libraryTagFilterResults: state.libraryTagFilterResults.map(book => updatedMap.get(book.id) ?? book),
  }));

  await get().fetchTagCounts();

  return { updated: books.length, changed };
},
```

```sql
create or replace function public.get_tag_counts_for_user()
returns table(tag_id text, book_count bigint)
language sql
security definer
set search_path = public
as $$
  select merged.tag_id, count(*)::bigint as book_count
  from (
    select distinct ul.id, tag_id
    from public.user_library ul
    cross join lateral jsonb_array_elements_text(
      coalesce(ul.book_data->'customTags', '[]'::jsonb) ||
      coalesce(ul.book_data->'autoTags', '[]'::jsonb)
    ) as tag_id
    where ul.user_id = (select auth.uid())
  ) merged
  group by merged.tag_id;
$$;

create or replace function public.get_books_by_tags(
  tags_to_filter text[],
  filter_by_favorites boolean default false
)
returns setof public.user_library
language sql
security definer
set search_path = public
as $$
  select ul.*
  from public.user_library ul
  where ul.user_id = (select auth.uid())
    and (
      coalesce(array_length(tags_to_filter, 1), 0) = 0
      or tags_to_filter <@ (
        select array_agg(distinct tag_id)
        from jsonb_array_elements_text(
          coalesce(ul.book_data->'customTags', '[]'::jsonb) ||
          coalesce(ul.book_data->'autoTags', '[]'::jsonb)
        ) as tag_id
      )
    )
    and (
      filter_by_favorites is not true
      or coalesce((ul.book_data->>'isFavorite')::boolean, false) is true
    )
  order by ul.created_at desc;
$$;
```
