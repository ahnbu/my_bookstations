# 제품 요구사항 정의서 (PRD)

---

### **1. 서비스 명**
- 마이 북스테이션 (My Bookstation)

### **2. 서비스 목표**
- **핵심 목표**: 사용자가 도서 정보를 쉽고 빠르게 검색하고, 지역 도서관의 재고 현황을 확인하며, 개인화된 도서 목록을 효율적으로 관리할 수 있도록 지원하는 원스톱 서비스 제공.
- **세부 목표**:
    - 사용자가 '전체', '제목', '저자', '출판사' 등 다양한 기준으로 도서 정보를 쉽게 검색할 수 있도록 한다.
    - 검색된 도서의 상세 정보(가격, 출판사, 저자, ISBN, 전자책 링크 등)를 명확하게 제공한다.
    - 경기도 광주시립도서관 및 광주 시립도서관 구독형 전자책의 도서 재고(소장 여부, 대출 가능 상태)를 실시간으로 확인한다.
    - 관심 도서를 '내 서재'에 추가/삭제하고, '추가순', '제목순', '저자순', '별점순', '읽음순'으로 정렬하여 개인 맞춤형 라이브러리를 구축할 수 있도록 한다.
    - '내 서재' 목록을 표준 데이터 형식인 CSV 파일로 내보낼 수 있는 기능을 제공하며, 도서관 재고, 전자책 재고, 추가일 등 상세 정보를 포함하고 파일명에 내보내기 날짜를 자동으로 추가한다.
    - 이메일/비밀번호 및 Google 소셜 로그인을 지원하여 사용자별 데이터 관리 기반을 마련한다.

### **3. 주요 기능 목록**
1.  **도서 검색**: 알라딘 Open API를 이용한 실시간 도서 정보 검색 ('전체' 검색 기본 제공).
2.  **도서 상세 정보**: 선택한 도서의 표지, 제목, 저자, 설명, 가격, 전자책 정보 등 상세 정보 표시.
3.  **도서관 재고 확인**: 경기도 광주시립도서관 및 광주 시립도서관 구독형 전자책 API를 연동한 실시간 재고 및 대출 가능 여부 확인.
4.  **내 서재**: 관심 도서를 저장하여 관리하는 개인화 기능.
    - Supabase DB에 로그인된 사용자 계정에 따라 데이터를 저장하여 여러 기기에서 동기화.
    - 도서 추가/삭제, 목록 정렬, 재고 갱신, 알라딘 페이지 연결, e북(시립구독) 링크 연결 등 기능 제공.
    - **읽음 상태 관리**: 각 도서의 읽음 상태('읽지 않음', '읽는 중', '완독')를 기록하고 변경할 수 있습니다.
    - **별점 평가**: 각 도서에 대해 1점에서 5점까지 별점을 매겨 개인적인 평가를 기록할 수 있습니다.
5.  **사용자 인증**: Supabase Auth를 이용한 이메일/비밀번호 및 Google 소셜 로그인 기능.
6.  **CSV 내보내기**: '내 서재' 목록 전체를 CSV 형식으로 다운로드하며, 전자책 재고 열을 포함하고 파일명에 내보내기 날짜를 자동으로 추가한다.
7.  **API 테스트**: 도서관 재고 조회 API를 ISBN과 제목으로 통합 테스트하는 개발자용 유틸리티 (UI 간소화).

### **4. 기술 스택 및 아키텍처**
- **UI/UX**: React, TailwindCSS
- **상태 관리**: Zustand
    - 중앙 집중식 스토어를 통해 컴포넌트 간의 결합도를 낮추고 상태 관리의 복잡성을 해소합니다.
    - 역할(인증, 도서, UI)에 따라 스토어를 분리하여 유지보수성과 예측 가능성을 높입니다.
- **인증 및 백엔드**: Supabase
    - 이메일/비밀번호 및 Google OAuth 인증을 처리합니다.
    - '내 서재' 데이터의 영구 저장을 위한 데이터베이스로 활용됩니다.
- **런타임 타입 안정성**: Zod
    - 외부 API(알라딘, 도서관)로부터 받는 데이터의 유효성을 런타임에 검증합니다.
    - 예기치 않은 데이터 형식으로 인한 앱의 충돌을 방지하여 서비스 안정성을 대폭 향상시킵니다.
- **서비스 계층**:
    - 외부 API 호출 로직을 별도의 서비스(Service) 파일로 분리하여 코드의 재사용성과 관심사 분리 원칙을 준수합니다.

### **5. 사용법**
1.  메인 화면 우측 상단의 '로그인' 또는 '회원가입' 버튼을 통해 계정을 관리합니다.
2.  메인 화면의 검색창에서 검색 유형('전체', '제목' 등)을 선택하고, 찾고 싶은 키워드를 입력한 후 검색 아이콘을 클릭합니다.
3.  검색 결과가 팝업 창에 나타나면, 원하는 **도서를 클릭**하여 선택합니다.
4.  선택한 도서의 상세 정보와 **경기도 광주시립도서관 및 광주 시립도서관 구독형 전자책의 재고 현황**이 화면 중앙에 표시됩니다.
5.  **'내 서재에 추가'** 버튼을 눌러 관심 도서를 하단의 '내 서재' 목록에 저장합니다.
6.  '내 서재' 섹션에서 저장된 도서 목록을 **정렬**하거나, 재고 정보를 개별적으로 **갱신**하고, 전체 목록을 **CSV 파일로 내보낼** 수 있습니다. CSV 파일에는 전자책 재고 정보가 포함되며, 파일명에 내보내기 날짜가 자동으로 추가됩니다. '내 서재' 목록의 도서 제목을 클릭하면, 제자리에서 상세 정보 모달이 나타나 해당 도서의 정보와 나의 독서 기록을 한눈에 확인할 수 있습니다.
